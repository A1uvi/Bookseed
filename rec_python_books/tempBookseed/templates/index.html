Model

Python accesses the description of the original book and all of the other options in a .csv file and converts them into vectors. Then, the sklearn cosin similarity midel is run to determine the similarity of all options to the original. Finally, the number output is used to sort the csv file to obtain the top 4 books with similar descriptions.

Front End

HTML files are used as templates, loaded when the url changes to display output. CSS formats this output and adds style, in addition to the loading animation. JavaScript is used to add more animations, such as text sliding in or elements popping up. Finally, JQuery is used to manage complexity and format the result of pressing buttons and going to HTML pages.

Flask

Flask connects the back-end python to cache in browser, whihc then calls HTML templates and loads CSS, JavaScript, and JQuery. Flask also allows for python o be used inline, which is how the library is made - python for loops inside the HTML file.

Product

The library allos one to view over 6,000 books with their description, ratings, and genre's. Clicking on one book will generate a page of reccomended similar books based off of the description. Additionally, a map of all the public libraries in America is given so that users may locate where they can get the book of their choosing fre of charge.
